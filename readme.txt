A plugin to display a custom announcement on the WordPress dashboard for all users.

== Description ==

This plugin allows you to display a custom announcement on your WordPress dashboard. It provides a simple way to communicate important information to your site's users directly from the dashboard.

== Features ==

* Display custom announcements on the WordPress dashboard
* Customize the appearance of the announcement
* Manage feedback and comments on the announcement
* Configure access levels for the announcement
* Integrate with Google Fonts for custom typography
* Add a "New Announcement" badge for recent announcements
* Customize the duration of the "New Announcement" badge
* Configure the announcement's text type and title font

== Frequently Asked Questions ==

= How do I customize the appearance of the announcement? =

You can customize the appearance of the announcement through the plugin settings, including text color, font type, and more.

= Can I restrict who can see the announcement? =

Yes, you can configure access levels for the announcement, allowing you to control who can view it.

= How do I manage feedback on the announcement? =

Feedback can be submitted and managed directly from the plugin's admin page. You can view, delete, and respond to feedback.

= How do I add a "New Announcement" badge? =

The "New Announcement" badge is automatically displayed for announcements that are new within a specified duration. You can configure this duration in the plugin settings.

= How do I change the font of the announcement title? =

You can change the font of the announcement title through the plugin settings, allowing for a more personalized look.

= How do I customize the text type of the announcement? =

You can customize the text type of the announcement (e.g., normal, bold, italic) through the plugin settings, providing flexibility in how your announcement is presented.

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `WP Admin Announcement` and click 'Install Now' or click on the `upload` link to upload `wp-admin-announcement.zip`
4. Click on `Activate plugin`
